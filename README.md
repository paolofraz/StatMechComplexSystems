# StatMechComplexSystems
Assignments of the course Statistical Mechanics of Complex Systems by prof. S. Suweis @ UniPD 2019 

## Outline

The course dealt with 4 practical assignments to be discussed during the oral exam. Inside each folder, the final report has been named as `Frazzetto_Paolo_ExX.pdf`.
* Ex1: Application of MaxEnt & Network Analysis - Barro Colorado Forest;
* Ex2: Determing the Spreading of Epidemics;
* Ex3: Neutral Theory and Emergent Patterns in Ecology;
* Ex4: Supervised Learning and Bayesian Inference.
